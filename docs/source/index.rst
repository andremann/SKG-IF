Welcome to SKG-IF documentation!
===================================

This documentation provides an overview of the work carried out by the RDA Interest Group on
`Open Science Graphs for FAIR Data <https://rd-alliance.org/groups/open-science-graphs-fair-data-ig>`_ 
towards the definition of the Scholarly Knowledge Graphs Interoperability Framwork (**SKG-IF**).

.. note::
   This project is under active development. 
   This documentation translates and will be in sync with the `discussion worksheet <https://docs.google.com/spreadsheets/d/1m8vQkklLWmDYk55oI6YxZcqkvL7CHp385stjIyy-5mI>`_ used for online discussion.
